# MstSohbetSistemi
SocketIO kullanılarak yapılmış bir flask projesidir. Eşzamanlı mesaj alışverişi sağlanmıştır.

# Used Python Library
- flask
- flask_sqlalchemy
- flask_socketio
- emoji

# Eksikler
- Kayıt ol
- Şifremi unuttum
- Admin paneli ayarları
En yakın zamanda güncellenecektir.
